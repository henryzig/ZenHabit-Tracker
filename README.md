# ZenHabit Tracker

[![license](https://img.shields.io/github/license/henryzig/ZenHabit-Tracker)](https://github.com/henryzig/ZenHabit-Tracker/blob/main/LICENSE)

## Overview

ZenHabit Tracker is a sleek and minimalist habit tracking application focused on helping users establish and maintain positive daily routines. Its distraction-free design encourages consistent habit formation with simple interactions and insightful progress views.

[View on GitHub](https://github.com/henryzig)

---

## Features

- Intuitive daily habit logging
- Visual progress charts and streak tracking
- Customizable habit reminders and notifications
- Dark mode support for reduced eye strain
- Export habit data to CSV
- Responsive design suitable for desktop and mobile

---

## Tech Stack

- **Frontend:** React.js with Tailwind CSS
- **Backend:** Node.js with Express
- **Database:** MongoDB Atlas
- **Authentication:** JWT-based auth system

---

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/henryzig/ZenHabit-Tracker.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd ZenHabit-Tracker
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Set up environment variables:**

   Create a `.env` file in the root directory and add your MongoDB connection URI and JWT secret.

   ```
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   ```

5. **Start the development server:**

   ```bash
   npm run dev
   ```

6. Open your browser and navigate to `http://localhost:3000`

---

## Usage

- Sign up or log in with your account.
- Add new habits you want to track.
- Mark habits as completed each day.
- View your progress and streaks on the dashboard.
- Export your habit data anytime via the settings page.
